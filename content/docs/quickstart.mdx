# Quickstart

Get your first AgentOS agent up and running in under 5 minutes.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ or Python 3.9+ installed
- An AgentOS API key (sign up at [agentos.dev](https://agentos.dev))
- Basic familiarity with JavaScript/TypeScript or Python

## Setup Your API Key

<Step number={1} title="Create environment file">
  Create a `.env` file in your project root:
  
  <CodeBlock language="bash" filename=".env">
    AGENTOS_API_KEY=your_api_key_here

## Create Your First Agent

<Step number={2} title="Initialize your agent">
  <Tabs defaultValue="typescript">
    <TabsList>
      <TabsTrigger value="typescript">TypeScript</TabsTrigger>
      <TabsTrigger value="python">Python</TabsTrigger>
    </TabsList>
    <TabsContent value="typescript">
      <CodeBlock language="typescript" filename="index.ts">
{`import { Agent } from '@agentos/sdk';

// Initialize the agent
const agent = new Agent({
  name: 'MyFirstAgent',
  model: 'gpt-4',
  systemPrompt: 'You are a helpful assistant that answers questions concisely.',
});

// Run a task
async function main() {
  const response = await agent.run({
    input: 'What is AgentOS?',
  });
  
  console.log(response.output);
}

main();`}
      </CodeBlock>
    </TabsContent>
    <TabsContent value="python">
      <CodeBlock language="python" filename="main.py">
{`from agentos import Agent

# Initialize the agent
agent = Agent(
    name="MyFirstAgent",
    model="gpt-4",
    system_prompt="You are a helpful assistant that answers questions concisely."
)

# Run a task
response = agent.run(
    input="What is AgentOS?"
)

print(response.output)`}
      </CodeBlock>
    </TabsContent>

</Step>

## Add Memory

<Step number={4} title="Enable memory for context">
  Make your agent remember previous interactions:

  <CodeBlock language="typescript" filename="memory-agent.ts">
{`import { Agent } from '@agentos/sdk';

const agent = new Agent({
  name: 'MemoryAgent',
  model: 'gpt-4',
  memory: {
    enabled: true,
    type: 'conversation',
  },
});

// First interaction
await agent.run({ input: 'My name is Alice' });

// Second interaction - agent remembers!
const response = await agent.run({ input: 'What is my name?' });
console.log(response.output); // "Your name is Alice"`}
  </CodeBlock>
</Step>

<TipCallout title="Pro Tip">
  Memory enables your agents to maintain context across conversations, making them more intelligent and personalized.
</TipCallout>
