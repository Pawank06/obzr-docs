# Installation

Install the AgentOS SDK in your preferred programming language and environment.

<InfoCallout title="System Requirements">
  Before installing AgentOS, ensure your system meets these requirements:
  - **Node.js**: Version 18.0 or higher
  - **Python**: Version 3.9 or higher
  - **Operating System**: macOS, Linux, or Windows
  - **Memory**: Minimum 4GB RAM recommended
</InfoCallout>

## Configuration

### API Key Setup

<Step number={1} title="Create environment file">
  Create a `.env` file in your project root:
  
  <CodeBlock language="bash" filename=".env">
{`AGENTOS_API_KEY=your_api_key_here
AGENTOS_API_URL=https://api.agentos.dev`}
  </CodeBlock>
</Step>

<Step number={2} title="Configure programmatically (optional)">
  Alternatively, configure the API key in your code:
  
  <Tabs defaultValue="typescript">
    <TabsList>
      <TabsTrigger value="typescript">TypeScript</TabsTrigger>
      <TabsTrigger value="python">Python</TabsTrigger>
    </TabsList>
    <TabsContent value="typescript">
      <CodeBlock language="typescript">
{`import { configure } from '@agentos/sdk';

configure({
  apiKey: process.env.AGENTOS_API_KEY,
  apiUrl: 'https://api.agentos.dev',
});`}
      </CodeBlock>
    </TabsContent>
    <TabsContent value="python">
      <CodeBlock language="python">
{`from agentos import configure

configure(
    api_key=os.getenv("AGENTOS_API_KEY"),
    api_url="https://api.agentos.dev"
)`}
      </CodeBlock>
    </TabsContent>
  </Tabs>
</Step>

## Troubleshooting

  <CardHeader>
